How It Works

Secure Authorization in 4 Steps

From user consent to AI transaction - every step is cryptographically secured and fully auditable.

01

User Creates Mandate

User opens the Mandate app and defines what the AI agent can do - cart management, purchase intents, or payments - with specific constraints.

02

Biometric Signature

User signs the mandate with drawn signature + Face ID/fingerprint. Hardware Secure Element creates cryptographic proof of consent.

03

AI Agent Acts

When the AI needs to transact, it presents the mandate. Your gateway verifies the cryptographic signature and checks constraints in real-time.

04

Verified & Logged

Every action is verified against mandate rules, logged with full audit trail, and stored for compliance reporting. User gets real-time notifications.
