# Agentic Commerce VR

AR/VR shopping experience for Meta Quest, Apple Vision Pro, and other VR platforms.

## Features (Planned)

- Immersive 3D product visualization
- Virtual showroom environments
- Gesture-based controls
- Spatial audio for voice assistant
- Virtual try-on for products
- Collaborative shopping experiences

## Platforms

- Meta Quest 2/3/Pro
- Apple Vision Pro
- HTC Vive/Focus
- PlayStation VR2

## Tech Stack

- React 360 / React Native VR
- Three.js for 3D rendering
- @react-three/fiber for React integration
- @react-three/xr for WebXR support

## Development

```bash
# Install dependencies
npm install

# Start development server
npm start
```
